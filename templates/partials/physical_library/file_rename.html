Rename Files
Clean up file names using metadata tags. Generates "Artist: Title" format, removes underscores, numbers, processing markers (PN, IDs) — keeps all metadata searchable in Rekordbox.
How it works

Input: Files like 918223_SomethingPN.mp3 or Artist_Remix_12345.mp3 or Track (2).mp3

Artist priority: Vocal/lead artist → Album artist/band → Producer. So a Donna Summer remix by Felix da-Housecat becomes Donna Summer: On the Radio (Felix da-Housecat remix) — using Donna as the primary artist.

Copy markers preserved: If a file is named Track (2).mp3, it becomes Artist: Track (2).mp3. Duplicates detected during rename get (2), (3), etc.

Process: Reads title and artist from ID3 tags with smart priority, preserves copy markers (2), (copy), (v2), etc., strips processing markers and odd patterns.

Output: Clean filenames like Artist: Title.mp3 or Artist: Title (2).mp3

Metadata preserved: Artist, album, and all other tags stay intact — full searchability in Rekordbox.
