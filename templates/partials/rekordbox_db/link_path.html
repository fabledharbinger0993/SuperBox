Relocate — Fix Broken Paths
Updates the database when you move or rename your music folders. RekordBox must be closed.
⤵ Drop old folder or type prefix and press Enter
⤵ Drop new folder or type / browse to the new root
⚠ RekordBox is open. Close it before relocating.
Link Playlists
Matches imported tracks to RekordBox playlists based on the folders they live in.
⤵ Drop folder here or type a path and press Enter
⚠ RekordBox is open. Close it before linking.
