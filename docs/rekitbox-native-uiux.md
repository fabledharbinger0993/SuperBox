# RekitBox-Native UI/UX Plan

## Library Browser
- Sidebar: Playlists, smart folders, search
- Main panel: Track list (sortable, filterable, editable)
- Track details: Inline editing, tagging, cue/loop editing

## Playlist Builder
- Drag/drop tracks into playlists
- Nested playlists (folders)
- Smart playlists (rules-based, e.g. BPM range, key, genre)
- Bulk actions: add/remove, tag, export

## Media Player
- Simple waveform or progress bar
- Play/pause, cue, seek, hotcue set
- Volume, pitch, keylock controls (optional)

## USB Export
- Export button: select drive, validate, write DB/files
- Progress and error reporting
- Option to backup before export

## Sync/Repair
- Button to re-import from Rekordbox DB
- Visual diff/merge for playlists, tracks, cues
- Warnings for schema drift or compatibility issues

---

This UI/UX plan supports a seamless, RekitBox-native workflow for library management, playlisting, and export, with robust safety and user feedback.
